# 📊 Measuring the Pulse of Prosperity: An Index of Economic Freedom Analysis

## 📌 Project Overview
This project analyzes the relationship between the Economic Freedom Index and a country’s overall economic prosperity. Using data visualization in Tableau, the project explores how different components of economic freedom influence indicators such as GDP per capita and development levels.

The main objective is to understand whether countries with higher economic freedom tend to achieve better economic outcomes.

---

## 🎯 Objective
- To analyze global Economic Freedom Index data
- To compare economic freedom with prosperity indicators like GDP per capita
- To identify trends and correlations between freedom and economic growth
- To build interactive dashboards using Tableau

---

## 📁 Dataset Information
The dataset includes:
- Country-wise Economic Freedom Index scores
- Sub-indicators such as:
  - Business Freedom
  - Trade Freedom
  - Investment Freedom
  - Property Rights
  - Government Spending
  - Corruption Level
- Economic indicators like:
  - GDP per capita
  - HDI (Human Development Index)

**Source:** Heritage Foundation / World Bank / UN Data

---

## 🛠️ Tools & Technologies Used
- Tableau (Data Visualization)
- Microsoft Excel (Data Cleaning & Preparation)
- CSV Dataset format

---

## 📊 Dashboards Created

### 🌍 Global Economic Freedom Map
- Visual representation of economic freedom scores by country
- Color-coded intensity for comparison

### 📈 Country Ranking Dashboard
- Top 10 and Bottom 10 countries based on economic freedom score

### 📉 Correlation Analysis
- Scatter plot showing relationship between Economic Freedom and GDP per capita

### 📊 Component Analysis
- Breakdown of different economic freedom indicators

---

## 🔍 Key Insights
- Countries with higher economic freedom generally show higher GDP per capita
- Strong correlation exists between property rights and economic performance
- Government intervention levels significantly impact economic freedom scores
- Developing nations show gradual improvement over time

---

## 📌 Project Outcome
This project successfully demonstrates how economic freedom influences prosperity and how data visualization tools like Tableau can be used to derive meaningful insights from global datasets.
